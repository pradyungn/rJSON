rJSON

A simple, fast JSON storage database written in Rust.

This is also the debut of dracula.css, a lightweight framework to style your thingy majiggy.

Commands

  • GET /api/{key}/[path] -> returns JSON data if available. Substitute path with context for your JSON. Ex: access {"a" : {...}} with /api/{key}/a, {...} will return. /api/{key} will return {"a": {...}}
  • POST /api/{key}/[path] -> send JSON data to the server. Same path system applies. Note that this overwrites preexisting data
  • DELETE /api/{key}/[path] -> Delete JSON data fro the server. No path deletes the file. Path replaces the relevant field with a Null value. If you request a specific field, and it is Null because of the DELETE, it will return an error.
